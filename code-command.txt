ຄຳສັງ github
 
1. github                   = ການເພິມໄຟຣ ທັງຫມົດ ເຂົ້າໃນ git
2.git comit -m "ຄຳສັງອະທິບາຍ" = ການ commit ເຂົ້າ git
3.git push                  = ເປັນການອັບໂຫຼດໂປເຈກ ຂຶ້ນ github
4.git pull                  = ເປັນການອັບເດດ ໂຄດລ່າສຸດຈາກ github
